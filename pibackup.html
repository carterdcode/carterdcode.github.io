Setting up a Local Backup Server on Raspberry Pi

Introduction

In this project, I will talk you through what I did to set up a local backup server on a Raspberry Pi to backup a Windows PC.

Requirements

  • Raspberry Pi
  • MicroSD card
  • External hard drive or USB drive
  • Windows PC
  • Ethernet cable

Steps

  1. Install Raspbian OS on Raspberry Pi
  2. Connect Raspberry Pi to the network
  3. Configure static IP address for Raspberry Pi
